Sequence Handling:

XString to store a single sequence BString

XStringSet for many sequences (kind of like a list) BStringSet DNAStringSet RNAStringSet AAStringSet

How do we go from a single sequence to a set and vice versa? # read the sequence as a set

zikaVirus <- readDNAStringSet(“data/zika.fa”) length(zikaVirus) # the set contains only one sequence [1] 1 width(zikaVirus) # and width 10794 bases [1] 10794

to collate the sequence use unlist:

zikaVirus_seq <- unlist(zikaVirus) length(zikaVirus_seq) # A 10794-letter “DNAString” instance [1] 10794 width(zikaVirus_seq) Error unable to find width for “DNAString”

to create a new set from a single sequence

zikaSet <- DNAStringSet(zikaVirus_seq, start = c(1, 101, 201), end = c(100, 200, 300)) zikaSet A DNAStringSet instance of length 3 width seq [1] 100 AGTTGTTGATCTGTGTGAGTCAGACT…AATTTGGATTTGGAAACGAGAGTTT [2] 100 CTGGTCATGAAAAACCCCAAAGAAGA…GTAAACCCCTTGGGAGGTTTGAAGA [3] 100 GGTTGCCAGCCGGACTTCTGCTGGGT…CAGCAATCAAGCCATCACTGGGCCT

length(zikaSet) [1] 3 width(zikaSet) [1] 100 100 100

Complement Sequence:

complement(a_seq) 12-letter “DNAString” instance seq: TACTAGAGCATT
